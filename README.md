# jetpack-fella
- Simple game written in raylib C
### Controls:
- jump - space
- move left - a + left
- move right - d + right 
### What you need:
- raylib
- gcc
### How to compile+run:
```
git clone https://github.com/drinkmonster/jetpack-fella
cd jetpack-fella
make
./jetpack-fella
```
